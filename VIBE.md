# VIBE

Minimal markdown workflow.

## Read order

1. `AGENTS.md`
2. `.vibe/STATE.md`
3. `.vibe/PLAN.md`
4. `.vibe/HISTORY.md` (optional)

## Workflow

- Define checkpoints in `.vibe/PLAN.md`.
- Track current work in `.vibe/STATE.md`.
- Implement one checkpoint.
- Record evidence.
- Open a PR
- Human review.

## Status values

- `NOT_STARTED`
- `IN_PROGRESS`
- `IN_REVIEW`
- `BLOCKED`
- `DONE`
